# Caudal Official Website

## Requirements

NPM & Hexo

``` bash
npm install hexo-cli -g
```

## Getting Started
Install dependencies as follow:

``` bash
$ git clone https://github.com/interwaremx/caudal-wiki.git
$ cd caudal-wiki
$ npm install
```
Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
